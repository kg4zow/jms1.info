# `jms1.info`

This repo contains the source code for the [`jms1.info`](https://jms1.info/) web site.

The pages are written in Markdown and converted to HTML using [mdbook](https://rust-lang.github.io/mdBook/). The [mdbook-template](https://github.com/kg4zow/mdbook-template) repo has more information about how this works, as well as the customizations I use on most of my sites.

## &#x2757;&#xFE0F; Github is NOT the primary server for this repo.

This repo's primary git server is not publicly accessible. I'm using Github as an alternate "push target", so people can *see* the repo if they're curious.

This is why **Github pull requests are disabled** for the repo. If you see something which needs to be corrected or improved, please [contact me directly](https://jms1.info/#contact).

I have a few different personal git servers, physically hosted in different parts of the world, using different software, including:

* [Gitolite](https://gitolite.com/) - very simple, no web interface, I've been using this for years.
* [Gitea](https://about.gitea.com/) - MUCH more feature-complete, and it has a web interface wth many of the same features as Github, Gitlab, and other "code hosting" platforms.

I also use [FOKS](https://foks.pub/) and [Keybase](https://keybase.io/) to store copies of my personal repos, as well as the primary copies of the repos I use for my servers' SSL keys.

The site's [Introduction page](https://jms1.info/#contact) has information about how to contact me.
